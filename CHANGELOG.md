# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1]
- **Navigation**: Added "Next Video" and "Previous Video" buttons to the player controls.

- **UI**: Added tooltips for new navigation buttons in English and Russian.
- **Documentation**: Updated technical documentation with new navigation signals and logic.
